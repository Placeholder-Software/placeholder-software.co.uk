It's our second asset after <a href="https://www.assetstore.unity3d.com/#!/content/70078?aid=1100lJDF">Dissonance Voice Chat</a> over a year ago!</p> <p>Wet stuff provides a powerful set of tools for making surfaces appear wet. Layered and animated decals can be used to simulate puddles, rain buildup, damp patches, leaking pipes, and more - all without making modifications to any existing texture or shaders. This allows you to dramatically change the atmosphere of a scene in seconds with simple to use tools in the editor.
