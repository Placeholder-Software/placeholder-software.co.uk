Wet Stuff

Version 1.1.0

We've added two new components intended to make Wet Stuff easy to integrate with a weather system. The new Puddle component will animate a puddle to realistically accumulate water as it is raining and then animate the puddle to slowly dry out once the rain stops. The new Drip Line component will automatically configure a particle system to drip from an edge such as a roofline when it's raining, creating tiny wet splashes where the particles impact. To control these components we've released a free new package on the asset store which connects together Enviro - Sky and Weather with Wet Stuff. The puddle and drip line components will automatically react to the Enviro weather as it changes between rain and sun.

In this release we have also dropped support for 2017.3, the lowest version we support for Wet Stuff is now Unity 2017.4 (LTS).

New Features

  • Added DripLine component
  • Added RainPuddle and AutoRainPuddle components
  • Added Enviro - Sky and Weather to automatically control weather related components

Bug Fixes

  • Fixed copy+paste not working for Wet Decal layer settings
  • Fixed wet particle splatter not spawning enough splatters when recycling is in use
  • Removed references to UnityEditor in some components, preventing projects which used them from building

  • HDRP is not supported. Please track progress here.
